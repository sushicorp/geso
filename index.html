Geso CSS Framework

Gesoは、日本語サイトを素早く書き始めるためのCSSフレームワークもどきです。

特徴
  • CSS @apply Ruleを利用した、1ファイルCSSフレームワーク（Chromeの「試験運用版のウェブ プラットフォームの機能を有効にする」フラグを有効にしないと動作しません）
  • スタイルの上書きが面倒くさいので、定義は最低限
